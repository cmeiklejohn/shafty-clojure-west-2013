what's shafty?

  • functional reactive programming "toolkit"
  • based on Flapjax
  • implemented in ClojureScript

flapjax

  • functional reactive programming
  • authored in JavaScript
  • core API / DOM API

shafty

  • implementation of the core API
  • meant to be a micro library
  • event streams and behaviours

example #1

    (->>
      (timer! 1000)
      (map! (fn [x]
        (.log js/console (str "Example 1 fired at " x)) x)))

example #2

    (def my-random-number
      (->>
        (timer! 1000 #(.random js/Math))
        (hold! 0)))

    (deref my-random-number)

    (def my-random-number-plus-one
        (lift! (fn [x] (+ x 1))) my-random-number))

    (deref my-random-number-plus-one)
