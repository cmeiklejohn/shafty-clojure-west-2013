what's shafty?

  • functional reactive programming "toolkit"
  • based on Flapjax
  • implemented in ClojureScript

flapjax

  • functional reactive progrmming language/library
  • authored in JavaScript
  • core API / DOM API

shafty

  • implementation of the core API
  • meant to be a micro library

example #1

    (defn timer []
      (-> (timer! 10000 (fn [] (js/Date.)))
          (map! (fn [x] (.log js/console "Timer ticked.") x)))))

    (-> (event! (get-element "save-button") "click")
        (map! (fn [x] (.log js/console "Button clicked.") x))
        (merge! (timer))
        (snapshot! (live-content))
        (map! build-request)
        (requests!))

example #2

    (defn timer []
      (-> (timer! 1000 (fn [] (js/Date.)))
          (map! (fn [x]
            (.log js/console (str "Ticked at: " x)) x))
          (hold! (js/Date.))))

    (-> (event! (get-element "reset-button") "click")
        (snapshot! timer)
        (map! (fn [x]
          (.log js/console (str "Clicked at: " x)) x))
        (hold! (js/Date.)))
